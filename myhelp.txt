★自作したショートカットファイル★

\C + b バッファリスト表示
\C + o アクティブウィンドウ切り替え
\C + ] ミニバッファコマンド(Exc + x と等価)
\C + x - j 指定行へ移動
\C + x - n 行番号表示/非表示 切り替え


★Ctrl + q に続けて押すコマンド★
a    全体を選択
c    コピー
e    右端折り返し切り替え
t    (C言語) タグを今のウィンドウで表示
\C-t (C言語) タグを別ウィンドウで表示
\C-b (C言語) タブジャンプ元へ戻る
r    置換
g    Grep
i    コンフィグファイルを開く( ~/.emacs.d/init.el)
h    自作ヘルプファイル(このファイルを開く)( ~/.emacs.d/myhelp.txt)
d    キーバインド一覧を表示(describe-bindings)



★コピー＆切り取り＆貼り付け★
\C + q - c       コピー
\C + q - \C + c  コピー
\C + x - \C + x	 切り取り
\C + v           貼り付け
